Recursos

La mejor forma de aprender desarrollo web es combinar teoría con práctica, y para eso necesitás buenas fuentes de información. Acá armamos una selección curada de recursos que usamos y recomendamos: desde la documentación oficial que todo dev debería tener marcada, hasta cursos en video, comunidades donde hacer preguntas, herramientas para experimentar y libros gratuitos. No es una lista exhaustiva — es una lista de lo que realmente vale la pena cuando estás empezando (y cuando ya tenés experiencia también).

Documentación Oficial

La documentación oficial es tu mejor amiga como desarrollador. A diferencia de tutoriales de terceros que pueden quedar desactualizados, la documentación oficial de cada tecnología es la fuente de verdad: está escrita por quienes crean y mantienen la tecnología, se actualiza con cada nueva versión, y cubre cada propiedad, método y parámetro con ejemplos. Al principio puede parecer densa o abrumadora, pero acostumbrarte a leerla es una de las habilidades más valiosas que podés desarrollar. Cuando te encuentres con un problema, la documentación oficial debería ser tu primera parada, no Google.

MDN Web Docs
La biblia del desarrollo web. Documentación completa de HTML, CSS, JavaScript y Web APIs, mantenida por Mozilla con ejemplos interactivos y guías para todos los niveles.
web.dev
Guías de Google sobre performance, accesibilidad, SEO y best practices de desarrollo web moderno. Los cursos de "Learn CSS" y "Learn HTML" son excelentes para principiantes.
Can I Use
Tabla de compatibilidad de funciones de HTML, CSS y JavaScript en todos los navegadores. Indispensable antes de usar cualquier feature nueva para verificar soporte.
W3C Standards
Las especificaciones oficiales del World Wide Web Consortium. Acá se definen los estándares de HTML, CSS, SVG y otras tecnologías web. Más técnico, pero es la fuente definitiva.
ECMAScript Specification
La especificación oficial de JavaScript (ECMA-262). Para entender cómo funciona el lenguaje debajo de todo. Nivel avanzado, pero fundamental para un dev senior.

Videos y Cursos

Los videos son una forma fantástica de aprender desarrollo web porque podés ver en tiempo real cómo alguien escribe código, explica conceptos y resuelve problemas. A diferencia de leer documentación, un buen video te muestra el flujo de trabajo completo: desde abrir el editor hasta ver el resultado en el navegador. La clave es elegir contenido de calidad y, sobre todo, no solo mirar — acompañar. Cada vez que el instructor escriba código, pausá el video y escribilo vos también. Acá compartimos algunos de los recursos en video que consideramos más útiles para aprender las bases de HTML, CSS y JavaScript.

HolaMundo
Aprende HTML ahora! Curso completo GRATIS desde cero
Curso completo de HTML en español, desde cero y sin conocimientos previos. Cubre todas las etiquetas esenciales, formularios, semántica y buenas prácticas con proyectos reales.
midudev
midudev
Uno de los mejores canales en español sobre desarrollo web. Contenido sobre HTML, CSS, JavaScript, React y herramientas modernas con explicaciones claras y dinámicas.
freeCodeCamp.org
freeCodeCamp.org
Cursos completos de varias horas sobre HTML, CSS, JavaScript y mucho más. Todos gratuitos y en inglés. Ideal para aprender en profundidad con proyectos prácticos de principio a fin.

Herramientas Online

Las herramientas online te permiten experimentar, probar y prototipar sin tener que instalar nada. Son perfectas para el aprendizaje porque podés escribir código, ver el resultado instantáneamente y compartirlo con otros. Algunas son playgrounds donde escribís HTML/CSS/JS directamente en el navegador, otras son utilidades especializadas que te ahorran tiempo en tareas repetitivas (minificar, formatear, generar colores, etc.). La ventaja de las herramientas online es que podés usarlas desde cualquier dispositivo con un navegador, y muchas permiten guardar o compartir tus experimentos mediante una URL.

MiniDevTools
Colección de 33 herramientas web gratuitas: playgrounds de Flexbox y CSS Grid, generadores de gradientes, sombras y border-radius, minificadores, formateadores y más. Todo en una sola página, sin instalar nada.
CodePen
El playground social para HTML, CSS y JavaScript más popular. Escribís código en tres paneles (HTML, CSS, JS) y ves el resultado en tiempo real. Ideal para experimentar y ver cómo otros resuelven problemas.
JSFiddle
Otro playground clásico para probar código web rápidamente. Interfaz más minimalista que CodePen, genial para pruebas rápidas y compartir snippets de código funcionales.
Figma
Herramienta de diseño web colaborativa basada en el navegador. Útil para diseñar layouts, prototipar interfaces y obtener medidas/colores exactos antes de maquetar. La versión gratuita es generosa.
PageSpeed Insights
Herramienta de Google para medir la performance de cualquier página web. Te da puntuaciones de Core Web Vitals (LCP, INP, CLS) y sugerencias específicas para mejorar la velocidad de carga.
Flexbox Froggy
Juego interactivo para aprender Flexbox guiando ranas a sus lilipads con propiedades CSS. 24 niveles progresivos que te llevan de justify-content a combinaciones avanzadas, todo mientras jugás. Disponible en español.
CSS Grid Garden
El hermano gemelo de Flexbox Froggy, pero para CSS Grid. Ayudá a un jardinero a plantar un jardín usando grid-column, grid-row, grid-area y más. 28 niveles que domás en una tarde, disponible en español.

Comunidades

Programar no es un deporte individual. Las comunidades son donde hacés las preguntas que Google no puede responder, donde compartís tus proyectos para recibir feedback, y donde te enterás de las últimas novedades del ecosistema. Además, participar en comunidades es una forma excelente de mejorar: al intentar explicar un problema o responder la pregunta de otro, consolidás tu propio conocimiento. No tengas vergüenza de preguntar — todos empezaron desde cero y las comunidades de desarrollo web son generalmente muy acogedoras con los principiantes. Solo acordate de buscar primero si tu pregunta ya fue respondida antes de postear.

Stack Overflow
El sitio de preguntas y respuestas más grande para programadores. Si tenés un error o un problema específico, es casi seguro que alguien ya lo preguntó y lo respondió acá. Es el primer lugar donde buscar soluciones.
r/webdev (Reddit)
Comunidad activa con más de 500K miembros. Ideal para seguir noticias del mundo web, pedir opiniones sobre tu código, compartir recursos y debatir sobre best practices y tendencias de la industria.
Discord (servidores de dev)
Muchas comunidades de desarrollo tienen servidores de Discord donde podés chatear en tiempo real con otros desarrolladores. Buscá servidores como "HolaMundo", "LatAmDev" o comunidades de tecnologías específicas.
DEV.to
Plataforma de blogs y artículos para desarrolladores, con una comunidad muy activa. Artículos prácticos sobre desarrollo web, tutoriales, opiniones y guías. También es un excelente lugar para empezar a escribir sobre lo que aprendés.

Libros y Lectura Gratuita

Los libros ofrecen una profundidad que difícilmente encontrás en tutoriales cortos o videos. Mientras que un video te muestra cómo hacer algo específico, un buen libro te explica el porqué: los fundamentos, el contexto histórico, las decisiones de diseño detrás de cada tecnología. Esto te da una base sólida que te permite adaptarte a nuevas tecnologías con más facilidad. Los libros de esta lista son gratuitos y están disponibles online, la mayoría en inglés. Si tu inglés no es perfecto, usá un traductor del navegador — el esfuerzo vale la pena por la calidad del contenido.

Eloquent JavaScript
De Marijn Haverbeke. Un clásico gratuito que va desde los fundamentos de JavaScript hasta temas avanzados como programación orientada a objetos, asincronía y Node.js. Incluye ejercicios prácticos en cada capítulo y un sandbox online para probar el código.
freeCodeCamp Curriculum
Plataforma interactiva con certificaciones gratuitas en desarrollo web responsive, JavaScript, algoritmos y más. Cada módulo combina teoría con proyectos prácticos que validás con tests automatizados. Es como un bootcamp gratutio.
You Don't Know JS (book series)
Serie de libros de Kyle Simpson que profundiza en los aspectos de JavaScript que la mayoría de desarrolladores pasan por alto. Temas como closures, this, prototipos, tipos y coerción. Nivel intermedio a avanzado, lectura obligatoria para entender JS de verdad.
CSS-Tricks
Uno de los blogs más influyentes sobre CSS y desarrollo web. Artículos detallados sobre técnicas de CSS, guías completas (la "Complete Guide to Flexbox" y "Complete Guide to Grid" son legendarias) y snippets reutilizables.
web.dev Learn
Cursos estructurados y gratuitos de Google: "Learn HTML", "Learn CSS" y "Learn JavaScript". Cada curso progresivo con explicaciones claras, ejemplos interactivos y quizzes para validar tu conocimiento.
